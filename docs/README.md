# Documentation

This directory gathers the documentation for the **Parallelium multilingual alignment dataset**.

The materials are organized by corpus and provide details on source texts, preprocessing choices, alignment methods, data structure, and corpus-specific editorial decisions.

## Contents

| Folder | Description |
|--------|-------------|
| [`dataset/bibles/`](dataset/bibles/) | Documentation for the scriptural corpus, including Biblical and Qur’anic alignment data, source materials, alignment guidelines, data structure, and statistics. |
| [`dataset/medieval_prose/`](dataset/medieval_prose/) | Documentation for the medieval prose corpus, including source materials, preprocessing choices, alignment strategy, and corpus-specific notes. |

## Note

Some source texts are not redistributed due to licensing restrictions. When relevant, the documentation indicates how to access or cite the original sources.

For a general overview of the project, see the main repository [`README.md`](../README.md).
